# how-to-create-gif-10-second-video-
This project shows how to create a GIF from screen recording for demonstrating project output.

## 🎯 Objective
To learn how to convert a screen recording into a GIF for showing project output.

## 🛠️ Tools Used
- Screen Recording Tool
- GIF Converter (online tool)

## 🚀 How it Works
1. Record the screen
2. Convert video into GIF
3. Use GIF in project documentation

## 📸 Output
GIF is used to showcase project demo in a simple way.

## 👩‍💻 My Contribution
Created and demonstrated the process of GIF creation from screen recording.
